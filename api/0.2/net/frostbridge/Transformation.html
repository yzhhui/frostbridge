net.frostbridge.Transformation

trait Transformation

sealed trait Transformation[Generated, Source]

 extends NotNull

Represents a general transformation of the result of a pattern into another pattern. (p: Pattern[S]) >>= (b: Binding[G, S]) processes the generated object of p and produces a new pattern to be matched for the unmarshalling case. For the marshalling case, it will do a reverse processing. This general case is like a flatMap/bind. The class to implement is Binding, which strips the generated pattern of top level AttributePatterns.
